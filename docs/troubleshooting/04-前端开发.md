# 前端开发问题

## 1. 仪表盘页面一直加载无内容

**现象**：进入仪表盘后页面持续转圈，没有内容出现。

**原因**：`DashboardPage` 使用 `adminGetStats().then(...)` 请求数据，但没有 `.catch()`。接口失败时 `stats` 状态永远是 `null`，页面一直停留在 loading 状态。

**解决**：加上错误处理，失败时显示提示而非永久 loading：
```tsx
useEffect(() => {
  adminGetStats()
    .then((r) => setStats(r.data))
    .catch(() => setError(true));
}, []);

if (error) return <div>加载失败，请刷新重试</div>;
if (!stats) return <LoadingSpinner fullPage />;
```
